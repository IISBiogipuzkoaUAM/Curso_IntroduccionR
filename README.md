# Curso_IntroduccionR
Curso introductorio sobre el uso de R (software libre y de código abierto) para el análisis de datos biomédicos.

Objetivos:
Formar a los participantes en el uso de R, desde el manejo básico del software (instalación, entorno de trabajo, paquetes), incluyendo el tratamiento completo de conjuntos de datos (importación, limpieza, manipulación y visualización), hasta la estadística aplicada, como análisis descriptivos, contrastes de hipótesis y regresión lineal. Se desarrollarán habilidades prácticas mediante casos de uso en contextos reales, aplicados a la biomedicina.
